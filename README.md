# dotfiles

My macOS dotfiles and setup script.

## Usage

Run `source <(curl -s https://raw.githubusercontent.com/cameronshemilt/dotfiles/main/setup.sh)`.

## Further Steps

2. Copy SSH Key in to `~/.ssh/`.
3. Copy AppStore signing certificate in to keychain.
